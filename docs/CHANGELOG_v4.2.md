# Betynz v4.2

- Replaced the public brand and PWA icons with the new Betynz lightning-B logo.
- Added install controls and automatic service-worker update checks. Browser confirmation is still required to install a PWA.
- Active matches now show LIVE, match minute and the current score on desktop, mobile and Z Fold layouts.
- Finished board picks settle as Won, Lost or Void even when they were provisional; verified-record status remains reserved for locked picks.
- Results and Recent Results now include all settled board picks and identify verified locks separately.
- Global coverage now keeps at least yesterday in the data window so completed picks can settle after midnight.
- Live-score refresh now checks the last two days.
- Rebuilt mobile and Z Fold drawer navigation with delegated page routing and stronger overlay layering.
