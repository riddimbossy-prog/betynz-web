# Betynz v5.4 — PPG Direction Agreement Gate

## Mandatory board-wide confirmation

Every Olympian, Zeus, Spartacus and Leonidas output is now checked against the same final PPG gate before publication.

- Home overall PPG is compared with away overall PPG.
- Home-at-home split PPG is compared with away-away split PPG.
- Both comparisons must identify the same stronger side.
- Missing, level or conflicting PPG directions return **No Bet**.
- A Home result selection is rejected when both PPG views point Away, and vice versa.
- Goals and BTTS markets still need the same-direction PPG confirmation before their own market rules are evaluated.
- The universal GG rule additionally requires both teams to exceed 1.50 PPG in both overall and split samples.

## Transparency

Match explanations now show overall PPG, venue PPG and the approved PPG direction. Existing settled historical records are not rewritten.

## Reliability correction

Null or empty numeric values are no longer converted to zero by the engine helper. Missing PPG evidence now remains missing and triggers abstention.
