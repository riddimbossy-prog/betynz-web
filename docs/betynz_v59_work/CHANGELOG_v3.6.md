# Betynz v3.6

## Mobile match layout
- Rebuilt phone and Galaxy Z Fold cover-screen match rows as separated cards.
- Teams and crests stay on the first line, the market and grade stay on the second line, and engine/confidence/odds stay in a dedicated footer.
- Add-to-slip control remains large and fixed at the top-right of each card.
- Improved truncation, spacing and touch targets for long team and league names.

## Deity artwork
- Increased deity visibility on engine cards, engine pick banners, dashboard and filtered engine pages.
- Preserved dark overlays so match text remains readable.

## Mobile navigation
- Increased bottom-navigation icon size and navigation height.
- Added a subtle active-icon background and more spacing between icon and label.
- Updated safe-area spacing for notches, browser bars and foldable cover displays.
